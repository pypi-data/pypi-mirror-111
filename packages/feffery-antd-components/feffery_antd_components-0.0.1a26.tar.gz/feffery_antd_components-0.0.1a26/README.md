# feffery-antd-components
`Plotly Dash`第三方组件库，将`Antd`中的常用组件带入`Dash`的世界🥳，最新测试版本：`0.0.1a25`

## 1 最新测试版本安装方式

```bash
pip install feffery-antd-components==0.0.1a25
```

## 2 已实现部件目录

- 日期选择框：`AntdDatePicker`
- 日期范围选择框：`AntdDateRangePicker`
- 分割线：`AntdDivider`
- 按钮：`AntdButton`
- 下拉选择：`AntdSelect`
- 树形控件：`AntdTree`
- 表格控件：`AntdTable`
- 锚点：`AntdAnchor`
